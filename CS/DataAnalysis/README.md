# Citizen Science Data Analysis

R scripts for analyzing and visualizing Lacs Sentinelles survey data.

## Scripts

| Script | Description |
|--------|-------------|
| `CSDataAnalysis1.Rmd` | Main analysis notebook |
| `algae-presence-figure.R` | Algae detection rates across surveyed lakes |
| `altitude-figure.R` | Altitude distribution of observations |
| `temporal-dist-figure.R` | Temporal distribution of survey submissions |
| `user-figure.R` | Volunteer participation analysis |
